# REST api server based on Node.js, express


## 사용 방법

```shell
  // npm 패키지 설치
  npm install

  // 1) mysql 설치
  // 2) DB 생성
  // 3) 생성한 DB에 맞게 createDBQuery.js 내 변수를 조정해준다.
  // 4) createDBQuery.js 실행해서 기본 테이블을 생성한다.
  node createDBQuery.js

  // 실행!
  npm start
```

`http://127.0.0.1:3000/` 서버를 실행한 후 아래 iOS 앱을 빌드해서 실행시킬 수 있다.
